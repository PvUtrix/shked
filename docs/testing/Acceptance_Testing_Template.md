# Шаблон Приемо-тестовых испытаний (ПТИ)
**На основании:** Roles_and_Functions_KTP_Variant_0.md
**Версия:** 1.1

## 1. Введение
Данный документ содержит набор тестовых сценариев для проверки функциональности системы "Шкед" (SmartSchedule) в соответствии с требованиями Variant 0.

## 2. Условные обозначения
- **ID**: Уникальный идентификатор тест-кейса.
- **Статус**:
    - ✅ **Pass** (Успешно)
    - ❌ **Fail** (Ошибка)
    - ⚠️ **Block** (Заблокировано)
    - ⏳ **Skip** (Пропущено)

---
