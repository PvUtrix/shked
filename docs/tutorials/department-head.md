# Руководство для Руководства кафедры (Сценарии использования)

В данном руководстве описаны основные сценарии работы для роли **Руководства кафедры** (Администратор кафедры / Заведующий кафедрой). Данная роль предназначена для высокоуровневого мониторинга учебного процесса, успеваемости студентов и работы преподавателей в рамках одной кафедры.

## Сценарий 1: Просмотр сводной статистики и расписания кафедры

**Цель:** Оценить текущую загрузку преподавателей, количество предметов и расписание экзаменов по кафедре.

1. Авторизуйтесь в системе под профилем **Руководства кафедры**.
2. По умолчанию откроется **Главная страница** вашей кафедры.
3. На дашборде вы увидите основные метрики: количество активных предметов, преподавателей и общую статистику успеваемости студентов.
4. Просмотрите ленту предстоящих экзаменов и зачетов, чтобы контролировать проведение аттестаций по вашим дисциплинам.

![Дашборд руководства кафедры со сводной статистикой](./assets/dept_head_dashboard.png)

## Сценарий 2: Мониторинг предметов и работы преподавателей

**Цель:** Ознакомиться со списком преподаваемых курсов, их лекторами и рабочими программами (РПД).

1. В боковом меню перейдите в раздел **Предметы**.
2. В списке отобразятся все дисциплины, закрепленные за вашей кафедрой.
3. Нажав на любую дисциплину, можно посмотреть назначенных преподавателей (лекторов, ассистентов) и загруженные учебно-методические материалы (например, рабочие программы дисциплин).
4. Чтобы детальнее изучить нагрузку сотрудников, перейдите в раздел **Преподаватели**.
5. Здесь вы увидите полный список сотрудников кафедры, количество часов/групп у каждого преподавателя и их средние показатели по проверке работ студентов.

![Список предметов и преподавателей кафедры с их нагрузкой](./assets/dept_head_subjects_lectors.png)

## Сценарий 3: Формирование отчетов и ведомостей

**Цель:** Получить агрегированную информацию по успеваемости и посещаемости для анализа или передачи в учебный отдел.

1. Откройте раздел **Отчеты** в боковом меню.
2. Для анализа посещаемости перейдите на вкладку **Отчеты по посещаемости**. Выберите нужный предмет или преподавателя, чтобы увидеть процент присутствия студентов на лекциях и семинарах.
3. Для контроля успеваемости перейдите на вкладку **Ведомости экзаменов** (или статистику успеваемости).
4. Здесь можно просматривать сводные результаты тестирований, экзаменов и зачетов по каждой дисциплине.
5. При необходимости сформированные отчеты можно экспортировать в сторонние форматы для распечатки или отправки.
