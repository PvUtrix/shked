# Инструкция по добавлению логотипа

## Шаги для добавления логотипа и favicon

### 1. Разместите файлы изображения

Поместите ваш логотип в папку `public/` со следующими именами:

- **`logo.png`** - основной логотип (рекомендуемый размер: 512x512px или 256x256px)
- **`icon.png`** - иконка для favicon (размер: 32x32px или 64x64px)
- **`favicon.ico`** - классический favicon (размер: 16x16px, 32x32px или 48x48px)

### 2. Форматы файлов

Поддерживаемые форматы:
- PNG (рекомендуется)
- SVG (для `logo.svg`, если предпочитаете векторный формат)
- ICO (только для `favicon.ico`)

### 3. Оптимизация

Для лучшей производительности рекомендуется:
- Использовать PNG с оптимизацией (например, через TinyPNG)
- Для SVG - минифицировать файл
- Для ICO - использовать инструменты конвертации PNG в ICO

### 4. Размеры файлов

Рекомендуемые размеры:
- `logo.png`: 512x512px или 256x256px (квадратное изображение)
- `icon.png`: 32x32px или 64x64px (для favicon в метаданных)
- `favicon.ico`: должен содержать несколько размеров (16x16, 32x32, 48x48)

### 5. Проверка

После добавления файлов:
1. Перезапустите dev-сервер Next.js
2. Проверьте отображение логотипа на всех страницах
3. Проверьте favicon во вкладке браузера

### Примечание

Если файлы не найдены, система автоматически использует fallback иконку `GraduationCap` из lucide-react.
