# Co-Lector (Со-преподаватель)

> Дополнительный преподаватель, ведущий занятия совместно с основным

## Описание роли

**Co-Lector** (Со-преподаватель) - роль для дополнительных преподавателей, которые ведут занятия по предмету совместно с основным преподавателем.

**Значение в системе**: `role = "co_lecturer"`

**Статус**: Добавлено в версии 2.0.0

## Права доступа

Co-Lector имеет те же права, что и Lector, но для своих предметов.

### ✅ Разрешено

- Все права [[Lector|преподавателя]]
- Создание домашних заданий
- Проверка работ студентов
- Отметка посещаемости
- Создание экзаменов
- Внесение результатов

### ❌ Запрещено

- Управление группами
- Создание предметов
- Создание расписания
- Управление пользователями
- Доступ к админ-панели

## Интерфейс

Co-Lector использует тот же интерфейс, что и [[Lector]].

**Главная страница**: `/co-lector` → redirect на `/lector`

## Модель данных

**Модель**: [[User]]  
**Роль**: `role = "co_lecturer"`

```typescript
interface CoLector extends User {
  role: 'co_lecturer'
  lectorSubjects: SubjectLector[]  // Привязка через SubjectLector с role='CO_LECTOR'
}
```

## Назначение со-преподавателя к предмету

**Действие**: Выполняет [[Admin]]

```typescript
await prisma.subjectLector.create({
  data: {
    subjectId: subjectId,
    userId: coLectorId,
    role: 'CO_LECTOR'
  }
})
```

## Связанные заметки

### Другие роли
